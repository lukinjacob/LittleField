# 📊 Data Scraper Dashboard

A browser-based automated data scraper that fetches transaction history from an API endpoint and downloads the results as numbered Excel (`.xlsx`) files on a configurable schedule.

**🌐 Live Site:** [https://YOUR-USERNAME.github.io/data-scraper/](https://YOUR-USERNAME.github.io/data-scraper/)

---

## 🚀 What It Does

- Fetches data from a REST API endpoint at scheduled intervals
- Converts the response into `.xlsx` Excel files
- Automatically downloads numbered files (e.g., `50.xlsx`, `51.xlsx`, ... `104.xlsx`)
- Runs entirely in the browser — no server, no Python, no installation
- Displays a live activity log and progress tracker
- Free to host on GitHub Pages
